Foundations

A short, foundational bibliography for Critical Code Studies as a method, together with key primary and secondary works behind the readings on this site.

The method

Critical Code Studies

Critical Code Studies (CCS) is an approach that applies critical hermeneutics to the interpretation of computer code, program architecture, and documentation within a socio-historical context. Mark C. Marino, 'Critical Code Studies' (2006)
  • Marino, M. C. (2006) 'Critical Code Studies', electronic book review, 4 December. The founding manifesto
  • Montfort, N., Baudoin, P., Bell, J., Bogost, I., Douglass, J., Marino, M. C., Mateas, M., Reas, C., Sample, M. and Vawter, N. (2013) 10 PRINT CHR$(205.5+RND(1)); : GOTO 10. Cambridge, MA: MIT Press. Collaborative single-line close reading
  • Marino, M. C. (2020) Critical Code Studies. Cambridge, MA: MIT Press. The book-length statement of the field
  • Marino, M. C. and Douglass, J. (2023) 'Introduction: Situating Critical Code Studies in the Digital Humanities'. Field overview
  • Berry, D. M. (2011) The Philosophy of Software: Code and Mediation in the Digital Age. Basingstoke: Palgrave Macmillan. Code, mediation, and critical theory
