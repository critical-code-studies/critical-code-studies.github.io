Readings

The programs, read closely

Each reading takes a single historic program and reads its source in depth: the code, the language it is written in, the machine it ran on, and the world it imagined. The list grows as readings are completed.

MIT AI Lab · 1968–70 · MacLisp / Micro-Planner SHRDLU

Terry Winograd's natural-language-understanding program for a simulated tabletop of blocks. A showpiece of symbolic AI: a full close reading of its code, its famous demonstration dialogue, its microworld, and the critique that followed.

Live · read now MIT · 1962 · PDP-1 assembly Spacewar!

Steve Russell, the Tech Model Railroad Club, and the PDP-1 in Building 26. The first widely shared computer game, read through its assembly listing, its hand-built physics, and the vector display that made it move.

In preparation MIT · 1966 · SLIP / MAD ELIZA

Joseph Weizenbaum's DOCTOR script, running on the SLIP list-processing language. The founding demonstration of Critical Code Studies: Mark C. Marino's reading takes the conversational illusion apart line by line. See also the book Inventing ELIZA (MIT Press).

Proposing a reading

Suggest a program

The readings are open-ended. If there is a program whose source you think deserves a sustained critical reading, and whose code survives in a form we can work from, we would like to hear about it. The strongest candidates are systems where the source is recoverable, the historical record is rich, and the code itself carries arguments worth drawing out.

Get in touch via the people page, or open an issue on the GitHub organisation.
